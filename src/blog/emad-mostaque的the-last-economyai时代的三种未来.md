大伙还记得Stable Diffusion的创始人Emad吗，他带着全新的创业项目Intelligent Internet（简称II）卷土重来了，就在刚刚，他官宣了II的旗舰产品“II Agent V1”。


![图片](/images/blog/emad-mostaque的the-last-economyai时代的三种未来-0.png)


看罢产品介绍，说实话，我并没有被惊艳到，Agent V1看上去有点像AI版的"要你命3000"，功能堆得挺多，但看不出什么统一的美学或者清晰的用户场景。

不过这种反差本身就挺有意思的。一个曾经做出Stable Diffusion这种划时代产品的人，现在拿出的新东西却平平无奇。

这让我好奇心起来了：这人到底在想什么？

于是我去翻了翻II的官网，发现了一些更有意思的信息。

其实Emad在去年写了一本书，书名是《The Last Economy》，而且这本书是开源的，可以在II官网免费下载。产品的灵光可能时有时无，但书里的想法却相当激进，甚至可以说，那款让我无感的II Agent V1产品，可能只是这些想法的一个脚注。


![图片](/images/blog/emad-mostaque的the-last-economyai时代的三种未来-1.jpg)


说实话，刚看到书名的时候我是有点怀疑的。毕竟写这种"未来学"书的人太多了，大部分都在贩卖焦虑。但Emad这个人有点不一样，他不是那种纸上谈兵的学者，他确实在AI这个行业里干过，而且干得还挺大。

在创办Stability AI之前，他是对冲基金的分析师。换句话说，他曾经是那个"靠预测吃饭"的人。后来他自己下场做Stable Diffusion，把闭源的图像生成模型彻底开源了，直接搅黄了Midjourney和DALL-E的好生意。


![图片](/images/blog/emad-mostaque的the-last-economyai时代的三种未来-2.jpg)


然后有趣的事情发生了：2024年，他被自己创办的公司踢了出来。

官方说法是"管理问题"，但懂行的人都知道，他开源的决定动了太多人的蛋糕。你想，如果AI模型都开源了，那些靠API收费的大公司怎么赚钱？所以Emad写这本书，某种程度上也是在对那些把他赶走的人说：你们赢了这一局，但长远来看，开源的趋势你们挡不住。

每次技术革命都有人喊"狼来了"，但最后狼都没来。汽车发明了，马车夫确实失业了，但工厂需要更多工人；电脑普及了，打字员消失了，但程序员成了新贵。那Emad凭什么说这次不同？

他说，因为以前的机器替代的是"手"，这次替代的是"脑"。

这个区别有多大呢？Emad算了一笔账：一个美国白领，公司要为他付工资、医保、办公室租金、咖啡机折旧……全部加起来，相当于每小时70美元的成本。而一个AI，完成同样的脑力工作，电费加算力成本大概是每小时0.36美元。两百倍的差价。

Emad在书里讲了一个特别损的例子。他说，按照现在的GDP算法，一个人得了癌症，对国家经济是好事，因为治疗要花钱，化疗、手术、住院，每一笔钱都计入GDP增长。但如果有人发明了治愈癌症的药，GDP反而会下降，因为以后没有治疗收入了。

所以他提出了一个新的衡量体系，叫MIND框架 ，四个维度：Material（物质），食物、能源、原材料；Intelligence（智能），知识、教育、AI能力；Network（网络），人际关系、社会信任；Diversity（多样性），生物和文化多样性。

这四个是相乘的关系，不是相加。如果任何一个接近零，其他再高也没用。比如一个社会物资极大丰富，但人人都不信任彼此，那文明健康度就是零。或者技术很发达，但生物多样性崩溃，那也是零。

Emad认为我们人类未来面临三种选择：

第一个是数字封建主义（Digital Feudalism），基本就是延续现在的趋势。几家科技巨头控制所有的AI基础设施，就像中世纪的地主控制土地一样。普通人呢？领一份"全民基本收入"，饿不死，但也别想着往上爬了。Emad原话挺扎心的："足够生存，不足以产生影响。"

你想想，现在我们的社会是不是已经有点这个味道了。你离不开微信，离不开支付宝，离不开ChatGPT。它们免费给你用，但你的数据、你的注意力、你的消费习惯，全成了它们的资产。AI时代只是把这个逻辑推到极致，你连"干活换取尊严"的资格都没有了，因为机器比你干得好还便宜。Emad说这是"舒适的牢笼"，我觉得形容得挺准。就像马尔库塞所说，牢笼不一定是铁窗，也可以是“舒适的不自由” 。

第二条路更糟，就是大分裂（The Great Fragmentation）。各国一看，AI这么厉害，得，闭关锁国吧。美国搞一套AI系统，中国搞一套，欧盟搞一套，谁也不兼容。然后大家开始"算法冷战"，不是用核武器互相威胁，而是用算法互相渗透、互相封锁。

Emad警告说，这种情况下，互联网会彻底分裂。你现在能上Google也能上Baidu，以后可能不行了。不同的AI系统会给你完全不同的"现实版本"：中国的AI告诉你一件事，美国的AI告诉你完全相反的另一件事，而且两边都有完美的证据链。到时候连"真相"都没有了，只有"你们家的真相"和"我们家的真相"。

第三条路是Emad真正想推的，也是我觉得最像科幻小说的那条路，叫共生系统（Human Symbiosis）。简单说，就是每个人都配一个AI。不是现在这种ChatGPT，而是一个真正懂你的、跟你一起成长的"数字双胞胎"。它帮你处理繁琐工作，你专注在创造性和有乐趣的事情上。钱怎么算？Emad发明了一个概念叫"Foundation Coin"，大概是说钱应该因为你是人而发给你，而不是因为你干了活。

说实话，读完这本书我心情挺复杂的。

一方面，Emad的诊断是对的。你看DeepSeek最近的表现，还有Claude、Gemini迭代的速度，说"脑力劳动正在贬值"并不是危言耸听。我身边已经有朋友的公司开始用AI替代初级文案和设计了，不是"辅助"，是"替代"。

按照目前的剧本走下去，我们确实是在往《赛博朋克2077》的方向一条道走到黑：巨头垄断、数据殖民、普通人领低保混日子。数字封建主义不是"会不会发生"的问题，是"已经发生了多少"的问题。

但另一方面，我对Emad开的药方也没那么乐观。

他书里描绘的"共生系统"听起来很美好，每个人都可以有一个AI双胞胎，钱按人头发而不是按劳分配。可问题是，谁来决定这个"Foundation Coin"怎么发？那些现在掌握算力的人，凭什么突然良心发现，把财富分散给每个人？

更讽刺的是，他自己做的那个II Agent V1，在我看来恰恰是他批判的那种方向：一堆功能堆在一起，看不出清晰的用户价值，更像是技术炫耀而不是真正解决问题。如果连他自己都做不出一个让人眼前一亮的"共生"产品，凭什么相信这个愿景能实现？

Emad在书里说，我们只有1000天的时间窗口做出正确的选择。但我觉得他可能高估了"选择"的空间。历史很少是设计出来的，更多是各种力量博弈后的意外结果。数字封建主义之所以成为默认路径，不是因为我们没想清楚，而是因为它是阻力最小的那条路。

所以读完这本书，我的感受更像是：他准确地描述了我们正在掉进的坑，但对于怎么爬出来，他的答案和他那个平庸的Agent产品一样，有理想，但缺乏说服力。

也许这就是开源理想主义者的宿命吧。他们能看穿系统的漏洞，甚至能写出漂亮的漏洞报告，但修bug这件事，从来都不是报告写得最好的人说了算。
