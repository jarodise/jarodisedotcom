# WeChat to Blog - Automated Publishing

Publish WeChat articles to your Astro-based blog with one command.

## Quick Start

```bash
source .venv/bin/activate && python .agents/skills/wechat-to-blog/scripts/auto_publish.py "<WECHAT_URL>"
```

Or with a manual title if auto-extraction fails:
```bash
source .venv/bin/activate && python .agents/skills/wechat-to-blog/scripts/auto_publish.py "<WECHAT_URL>" "Article Title"
```

That's it. The script will:
1. ✅ Scrape the article with proper formatting
2. ✅ Download all images to `public/images/blog/`
3. ✅ Generate blog post with correct frontmatter
4. ✅ Auto-select tags based on content
5. ✅ Build the site
6. ✅ Tell you the final URL

Then just commit and push:
```bash
git add . && git commit -m "Add new post: <title>" && git push
```

## What the Script Does

### 1. Scraping
- Extracts article title (or asks if not found)
- Extracts content with **proper paragraph breaks** preserved
- Identifies all images with their exact positions

### 2. Image Handling
- Downloads all images from WeChat CDN
- Saves to `public/images/blog/<slug>-0.png`, `<slug>-1.jpg`, etc.
- Inserts markdown image tags at correct positions

### 3. Auto-Generated Frontmatter
```yaml
---
slug: <auto-generated-from-title>
title: <exact-from-wechat>
description: <first-paragraph-auto-extracted>
date: <current-date>T12:00:00.000Z
author: 数字游民Jarod
tags:
  - <auto-selected-based-on-keywords>
featured: false
editable: true
---
```

### 4. Tag Selection
Tags are auto-selected based on content keywords:
- **科技AI**: AI, ChatGPT, 人工智能, code, algorithm...
- **思考哲学**: 哲学, 思考, 价值, 社会...
- **数字游民**: 远程, 旅居, 签证, nomad...
- **生活方式**: 生活, 健康, 习惯...
- **创业赚钱**: 创业, 商业, 收入...
- **自我成长**: 成长, 学习, 读书...
- **旅行旅居**: 旅行, 目的地, 城市...
- **英语学习**: 英语, IELTS, 口语...

## Manual Workflow (If Needed)

If the auto script fails, use manual workflow:

### 1. Scrape
```bash
source .venv/bin/activate && python .agents/skills/wechat-to-blog/scripts/scrape_wechat.py "<WECHAT_URL>"
```

### 2. Check Output
```bash
cat /tmp/wechat_content.txt
```

### 3. Download Images
```bash
curl -sL "<image-url>" -o public/images/blog/<name>.png
```

### 4. Create Blog Post
Copy content from `/tmp/wechat_content.txt`, replace `IMAGE_URL_X` placeholders with actual paths.

## Troubleshooting

### "Could not extract title"
WeChat loads title via JavaScript which can be unreliable. Solution:
```bash
python auto_publish.py "<URL>" "Your Article Title"
```

### Images at wrong positions
The auto script handles this. If doing manually: **Do not guess** - use `IMAGE_URL_X` placeholder positions exactly as they appear in scraped content.

### Build errors
```bash
npm run build
```
Check for:
- Missing image files
- Malformed frontmatter
- Special characters in title
